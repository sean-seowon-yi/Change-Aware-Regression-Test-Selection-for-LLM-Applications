# Change-Aware Regression Test Selection for LLM Applications

**Short name:** change-aware LLM test selection (CARTS-style RTS for prompts)

Typed change analysis, impact prediction, and conservative test subset selection so teams can skip rerunning full eval suites when only part of a prompt, config, or model surface changes.

See [`docs/PROPOSAL_SUMMARY.md`](docs/PROPOSAL_SUMMARY.md) for the full problem formulation, design, and evaluation plan.
