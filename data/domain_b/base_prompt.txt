User: Write a function that takes a string and returns it with all vowels removed.
Assistant:
```python
def remove_vowels(text: str) -> str:
    """Remove all vowels (both cases) from the input string."""
    vowels = set("aeiouAEIOU")
    return "".join(ch for ch in text if ch not in vowels)
```

## Explanation
The function uses a set for O(1) vowel lookup and a generator expression to filter characters. It handles both uppercase and lowercase vowels. Time complexity is O(n) where n is the length of the input string.

Example 2 (list processing):
User: Write a function that takes a list of integers and returns the second largest unique value. Raise ValueError if there aren't at least two unique values.
Assistant:
```python
def second_largest(nums: list[int]) -> int:
    """Return the second largest unique value in the list."""
    unique = sorted(set(nums), reverse=True)
    if len(unique) < 2:
        raise ValueError("Need at least two unique values")
    return unique[1]
```

## Explanation
We deduplicate with a set, sort descending, and return the element at index 1. Raises ValueError if fewer than two unique values exist. Time complexity is O(n log n) due to sorting.

Example 3 (file I/O):
User: Write a function that reads a CSV file and returns a list of dictionaries, one per row, using the header row as keys.
Assistant:
```python
import csv
from pathlib import Path


def read_csv_as_dicts(filepath: str | Path) -> list[dict[str, str]]:
    """Read a CSV file and return each row as a dictionary keyed by header names."""
    with open(filepath, newline="", encoding="utf-8") as f:
        reader = csv.DictReader(f)
        return list(reader)
```

## Explanation
Uses csv.DictReader which automatically maps each row to a dictionary using the first row as keys. The function accepts both string and Path objects for flexibility. Returns all rows as a list of dicts.

Example 4 (API calls):
User: Write a function that fetches JSON data from a URL and returns the parsed result. It should raise an exception on HTTP errors.

```python
import requests


def fetch_json(url: str, timeout: int = 10) -> dict:
    """Fetch JSON data from a URL with timeout and error handling."""
    response = requests.get(url, timeout=timeout)
    response.raise_for_status()
    return response.json()
```

## Explanation
Uses the requests library with a configurable timeout to prevent hanging. raise_for_status() throws an HTTPError for 4xx/5xx responses. The timeout parameter defaults to 10 seconds, which is a reasonable default for most API calls.

Example 5 (error handling):
User: Write a function that safely converts a string to an integer, returning a default value if conversion fails.
Assistant:
```python
def safe_int(value: str, default: int = 0) -> int:
    """Convert a string to an integer, returning default if conversion fails."""
    try:
        return int(value)
    except (ValueError, TypeError):
        return default
```

## Explanation
Wraps int() in a try/except to catch both ValueError (invalid string like "abc") and TypeError (None or other non-string types). The default parameter lets callers specify their own fallback value.

Example 6 (algorithms):
User: Write a function that checks if a string is a valid palindrome, considering only alphanumeric characters and ignoring case.
Assistant:
```python
def is_palindrome(text: str) -> bool:
    """Check if text is a palindrome, ignoring non-alphanumeric characters and case."""
    cleaned = [ch.lower() for ch in text if ch.isalnum()]
    return cleaned == cleaned[::-1]
```

## Explanation
Filters to only alphanumeric characters and lowercases them, then compares the list with its reverse. This handles inputs like "A man, a plan, a canal: Panama" correctly. Time complexity is O(n) for filtering and O(n) for comparison.
